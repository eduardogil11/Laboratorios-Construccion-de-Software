¿Cuál es la diferencia entre Internet y la World Wide Web?

La diferencia entre Internet y la World Wide Web es que el internet es una enorme red goblal con muchas redes y la World Wide Web es aquella que recopila información por medio del internet.

¿Cuáles son las partes de un URL?

Protocolo HTTPS, Subdominio, Dominio, TLD, Ruta, Parámetro, Etiqueta.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Su función es recopilar datos de un recurso.
HEAD: Su función es recopilar datos de un recurso pero no cuenta con el cuerpo de la respuesta.
POST: Su función es mandar una entidad a un recurso.
PUT: Su función es remplazar las representaciones actuales de un recurso.
PATCH: Su función es realizar modificaciones parciales a un recurso.
DELETE: Su función es borrar un recurso

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

POST, ya que manda entidades al recurso.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

GET.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Se ignifica que fue exitosa la operación, así que no cuenta con algún error.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

No, ya que el error puedo haber sido de lo contrario, de lo contrario es responsabilidad del desarrollador corregirlo.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Si, ya que es un error en la programación.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

Que no existen en HTML5 y algunos elementos de HTML 4 que en HTML5 estén desaprobados son profile, longdesc, nohref, rev y charset.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

Las diferencias principales entre HTML 4 y HTML5 son que HTML5 incluye mejor soporte y diseño en páginas web y dispositivos y que HTML4 es mucho más desordenado.

¿Qué componentes de estructura y estilo tiene una tabla?

Filas, columnas, encabezados, contenido de tabla, títulos de columnas y posiciones en la tabla.

¿Cuáles son los principales controles de una forma HTML5?

¿Cuál es el ciclo de vida de los sistemas de información?

Fase de planificación, fase de análisis, fase de diseño, fase de desarrollo, fase de integración y periodo de pruebas, fase de implementación, y fase de mantenimiento.

¿Cuál es el ciclo de desarrollo de sistemas de información?

Planificación del concepto, definir los requisitos, el diseño,fase de desarrollo y pruebas, puesta en marcha, operaciones y mantenimientos, y disposición.
