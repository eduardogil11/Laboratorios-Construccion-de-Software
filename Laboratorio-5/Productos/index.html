¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos, ya que esto impide que los datos inválidos lleguen al servidor.

¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Para saltarte la seguridad de validaciones hechas con JavaScript simplemente se escribe (novalidate) en el inspector de la página.

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

Es una buena práctica, ya que la validaciones de JavaScript se usan para la verificación del usuario que no haya cometido algún error.
