¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Las diferencias entre Java y JavaScript es que Java es un lenguaje complicado y JavaScript es un lenguaje interpretado, al igual que Java se recomienda mucho para back-end y JavaScript para front-end y que Java utiliza más recursos que JavaScript.

Las semejanzas entre Java y JavaScript es que comparten nombres entre si y que tienen una sintaxis muy similar.

¿Qué métodos tiene el objeto Date?

Date.getDate(), Date.getDay(), Date.getFullYear(), Date.getHours(), Date.getMilliseconds()

¿Qué métodos tienen los arreglos?

pop(), push(), splice(), indexOf(), shift().

¿Cómo se declara una variable con alcance local dentro de una función?

Una variable con alcance local dentro de una función se declara con var, let, const.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Utilizar variables globales dentro de funciones implica que solo se podran utilizar dentro de la función declarada.
